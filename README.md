遠端高齡者視覺健康檢查系統 (Elderly Wellness Checker) 這是一個基於 Python 與電腦視覺技術的健康檢查工具，專為遠端監測高齡者健康狀況而設計。系統透過分析臉部特徵與舌頭狀態，提供即時的健康預警。

🌟 核心功能

臉部狀態分析 (face_analyzer.py) 疲勞偵測：利用眼睛縱橫比 (EAR) 計算眨眼頻率與眼睛開啟程度，判別使用者是否過度疲勞。
皮膚與唇色分析：偵測臉頰與嘴唇的顏色變化（如：蒼白、發紅、發紺），反映潛在的血液循環或健康問題。

表情張力監測：分析眉毛與嘴部的特徵點，識別使用者的不適感或表情緊繃程度。

對稱性檢查：針對臉部關鍵點進行對稱性評分，輔助篩查中風等突發性症狀。

舌象中醫輔助分析 (tongue_analyzer.py)
舌質顏色：識別舌頭是否呈現蒼白、偏紅或青紫 。

苔層厚度：分析舌苔的覆蓋程度，分為薄苔、中等或厚苔 。

潤澤度監測：透過反光與紋理分析，判別舌面水分狀態（如：乾燥或濕潤） 。

自動化匯報系統 (wellness_checker.py) 即時標註：在影像上直接標示分析結果與警示訊息。
健康摘要：自動生成包含「警報 (Alerts)」與「警告 (Warnings)」的 JSON 報告。

雙模式運行：支援單張照片分析或攝像頭即時監控模式。

🛠 安裝與環境設定 系統需求 Python 3.9+

📊 技術架構

MediaPipe: 用於高精度的臉部 468 個特徵點追蹤 。

OpenCV: 處理影像擷取、色彩空間轉換及繪圖標註 。

NumPy: 負責複雜的顏色比率與對稱性數學運算 。

免責聲明：本系統僅作為居家健康輔助參考，分析結果不能代替專業醫療診斷。若感到身體不適，請務必諮詢專業醫療人員。
